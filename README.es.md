## `> whoami` — Diseñador Visual que aprendió a programar

Soy **desarrollador de Chrome extensions y diseñador visual** con base en Girona, España.
Pasé años al frente de **Projekta2 Creative Studio** — experiencias web inmersivas donde diseño e ingeniería eran indistinguibles. Ese background da forma a todo lo que construyo.

> *Construyo herramientas para **la tercera semana de uso** — cuando la novedad ha desaparecido y solo queda si te ahorra tiempo.*

**Lo que diferencia mis herramientas:**

| Principio | Qué significa |
|-----------|--------------|
| 💳 **Pago único** | Pagas una vez. Es tuyo para siempre. Sin impuesto de suscripción por funciones que no deberían ser recurrentes. |
| 🔑 **BYOK — Trae Tu Propia Clave** | Tus API keys no salen de tu máquina. Cero proxy en backend. Cero recogida de datos. |
| 🌍 **Bilingüe desde el día uno** | EN/ES integrado en la arquitectura i18n — no traducido después. |
| 🚫 **Telemetría cero** | Ninguna. Ni siquiera error reporting a menos que hagas opt-in explícito. |
| 🌗 **Dark mode de verdad** | Todas las extensiones salen con dark mode desde la v1. No es una casilla — es una decisión de diseño. |
