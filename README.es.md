## `> whoami`

Soy Diseñador Visual que cruzó al código — no por accidente, sino porque **las herramientas que necesitaba no existían.**

Pasé años dirigiendo **Projekta2 Creative Studio**, construyendo experiencias web inmersivas en la intersección entre diseño e ingeniería: del tipo en que no estabas seguro si mirabas una web o una instalación.

Ese bagaje define todo lo que publico:
- **Obsesión por el detalle** — dark mode que funciona de verdad, espaciado que respira, interacciones que se sienten intencionales
- **Bilingüe desde el primer día** — EN/ES integrado en la arquitectura, no pegado después
- **Anti-dark-patterns** — sin urgencia falsa, sin freemium trampa, sin telemetría

Ahora construyo **extensiones de Chrome** para freelancers y desarrolladores, y **recursos open source** para la comunidad.

## `> cat ~/philosophy`

<p align="center">
  <img src="assets/quote.svg" alt="La mayoría de herramientas se construyen para impresionar en la demo. Yo construyo para la tercera semana de uso." width="830" />
</p>

<br />

**Lo que eso significa en la práctica:**

- 💳 **Pago único** — pagas una vez, es tuyo. Sin suscripción para funciones que no deberían ser recurrentes.
- 🔑 **BYOK (Bring Your Own Key)** — tus claves de API se quedan en tu máquina. Sin proxy backend, sin recopilación de datos.
- 🌗 **Dark mode hecho bien** — no es un añadido tardío. Cada extensión incluye dark mode real desde la v1.
- 🌍 **Bilingüe desde el primer día** — EN/ES integrado en la arquitectura i18n, no traducido después.
- 🚫 **Sin telemetría** — ninguna. Ni siquiera informes de errores a menos que lo actives explícitamente.
